Web dasturlash

Web-dasturchi ( inglizcha: Web Developer ) - bu mijoz-server modeli yordamida Butunjahon o'rgimchak to'ri dasturlarini ishlab chiqishga ixtisoslashgan yoki u bilan shug'ullanadigan dasturchi. Ilovalar odatda mijozda HTML, CSS va JavaScriptni, serverda PHP, ASP.NET (C #), Python, Node.js, Go yoki Javani va mijoz va server o'rtasidagi aloqa uchun http serveridan foydalanadi. Veb-tarkibni boshqarish tizimi ko'pincha veb-ilovalarni ishlab chiqish va qo'llab-quvvatlash uchun ishlatiladi.

Kubik rubik

Rubik kubi (aslida "sehrli kub" , so'zlashuv tilidagi versiyasi - Rubik kubi osilgan . bűvös kocka ) - vengriyalik haykaltarosh va arxitektura o'qituvchisi Erno Rubik tomonidan 1974 yilda ixtiro qilingan (va 1975 yilda patentlangan) mexanik boshqotirma . Boshqotirma 3x3x3 o'lchamdagi plastik kub (original) bo'lib, unda 54 ta ko'rinadigan rangli stikerlar mavjud.
